# CRMEB-PC商城

## 👋 特点

- 一个电子商务购物网站
- 现实生活中的购物车和商店结账流程的模式
- 支持线上客服系统
- 页面logo/客服广告/轮播等自定义功能配置

## 🌴 环境配置

在项目根目录env.js中进行配置


## ⚡ 运行

```bash
$ cd product //项目目录
$ npm i
$ npm run dev
```

## 🎒 打包配置
```
// 测试环境
npm run build:test

// 生产环境
$ npm run build
```
